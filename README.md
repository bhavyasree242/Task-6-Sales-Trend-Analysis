# Task 6 - Sales Trend Analysis Using Aggregations

## Objective
Analyze monthly revenue and order volume using SQL aggregation functions.

## Dataset
- Orders.csv
- Details.csv

## Tools Used
- MySQL Workbench

## SQL Concepts Used
- JOIN
- GROUP BY
- ORDER BY
- SUM()
- COUNT()
- COUNT(DISTINCT)
- STR_TO_DATE()

## Analysis Performed
1. Imported Orders and Details datasets into MySQL.
2. Joined both tables using Order ID.
3. Converted Order Date from text to date format using STR_TO_DATE().
4. Extracted Year and Month from Order Date.
5. Calculated monthly order volume.
6. Calculated monthly revenue.
7. Identified the top 3 months by revenue.


## Outcome
Learned how to perform sales trend analysis using SQL aggregations, date functions, grouping, and sorting techniques.
